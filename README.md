# 🌐 Programación III: Desarrollo Web Full Stack (UTN)

Este repositorio es el espacio dedicado a la entrega y seguimiento de los trabajos prácticos de la materia **Programación III** de la Tecnicatura Universitaria en Programación a Distancia [cite: 5, 2025-11-28]. Aquí se documenta mi evolución desde las bases de la maquetación hasta el desarrollo de aplicaciones dinámicas.

## 🚀 Hoja de Ruta del Repositorio

A lo largo del cuatrimestre, este espacio cubrirá los siguientes pilares tecnológicos:

### 🔹 Fase 1: Estructura y Estilo (En progreso)
* **HTML5:** Maquetación semántica y accesibilidad.
* **CSS3:** Flexbox, Grid Layout, Variables CSS y metodologías de nombrado (BEM).
* **Responsive Design:** Adaptación de interfaces para dispositivos móviles.

### 🔹 Fase 2: Lógica de Programación (Próximamente)
* **JavaScript (Vanilla):** Manipulación del DOM, eventos y lógica en el navegador.
* **Consumo de APIs:** Uso de `fetch` para conectar aplicaciones con datos externos.

### 🔹 Fase 3: Backend y Persistencia
* **Node.js / Java:** Desarrollo del lado del servidor.
* **Bases de Datos:** Integración con sistemas de gestión de datos para aplicaciones reales.
